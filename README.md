The chatbot is an AI-based emotion-aware chatbot that understands the emotional tone of a user’s message and responds in a suitable conversational style.
Instead of giving plain responses, the chatbot adapts its tone such as friendly, sarcastic, mentor-like, or supportive, making the conversation feel more natural and engaging.The application provides responses in both text and voice format, creating a more interactive and human-like experience.

Features:
Emotion-aware conversation
Responds in different conversational styles:
    Friendly
    Sarcastic
    Mentor / Advisor
Generates textual responses
Provides voice replies
Interactive chatbot interface

Use Cases:
Emotion-aware conversational AI
Mental wellness or supportive chat systems
Human-like chatbot applications
